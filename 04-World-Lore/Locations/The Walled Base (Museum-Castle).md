# The Walled Base (Museum-Castle)

> ⚑ Proper name unknown — descriptive title. First seen: [[Session 01 — Graduation Day]].

An old **museum that was once a castle**, so it retains defensive walls. Home base of the party and the [[Ashwardens]] training program (boot camp).

## Known Features (S01)

- **Banquet hall** — round officers' table, recruit tables, large fireplace; hosted the graduation ceremony; displays the Ashfall Britannia seal
- **Ashwarden armory** — basic armor (leather, chainmail) and standard weapons
- **Main gate** — heavy iron, lever-operated, flanked by guard shacks (a spare lever is kept in the shack *outside* the wall); barracks near the gate
- **Back gate** — exists; [[Commander Varan]] fled through it
- Assigned seating for recruits; kitchen staff

## Events

- **S01:** Graduation banquet; vampire attack on the surrounding city; civilians trapped outside the closed main gate were saved by the party's sortie.
